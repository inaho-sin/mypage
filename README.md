# GitHub Pages 有効化 / 無効化 メモ

---

## 有効化の手順

1. GitHubでリポジトリを開く
2. 上部タブの **Settings**（設定）をクリック
3. 左サイドメニューの **Pages** をクリック
4. 「Build and deployment」の **Branch** を `main`（または `master`）に設定
5. **Save** をクリック
6. 数分後に以下のURLでページが公開される

```
https://ユーザー名.github.io/リポジトリ名/
```

> ※ 反映まで数分かかることがある。すぐ見られなくても少し待つ。

---

## 無効化の手順

1. GitHubでリポジトリを開く
2. 上部タブの **Settings**（設定）をクリック
3. 左サイドメニューの **Pages** をクリック
4. 「Build and deployment」の **Branch** を `None` に変更
5. **Save** をクリック

> ※ 無効化してもリポジトリのコードやコミット履歴は消えない。

---

## 注意点

- 無料プランでは **パブリックリポジトリのみ** 利用可能
- SSLは **自動で有効**（自分で設定不要）
- 一度公開すれば **自分で無効化しない限り停止しない**
- パスワードやAPIキーなど機密情報は含めないこと

---

## 公開後のURL形式

| 種類 | URL |
|------|-----|
| ユーザーページ | `https://ユーザー名.github.io/` |
| プロジェクトページ | `https://ユーザー名.github.io/リポジトリ名/` |
